### Setup
Install
```sh
  npm install
```

### Building
Build
```sh
  npm run build
```

Compile
```sh
  npm run compile
```

### Debugging
Build (Watch)
```sh
  npm run watch
```

Compile (Watch)
```sh
  npm run debug
```

### View Resume
**On MacOS**
```sh
  open -a "Google Chrome" resume.pdf
```
